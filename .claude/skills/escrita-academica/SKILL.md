# Escrita acadêmica do Pedro

## Os três anti-padrões (Pedro revisa linha a linha procurando por eles)

1. **Travessões (em-dashes): PROIBIDOS.** Pedro os chama de "a marca mais explícita de escrita feita por IA" e já removeu manualmente todos de um capítulo inteiro. Substituição padrão: **vírgula**. Só usar dois-pontos ou quebrar a frase se a vírgula ficar genuinamente ambígua. Isso vale para qualquer texto entregue a ele, inclusive rascunhos.

2. **Autorreferência em terceira pessoa: proibida em texto de primeira pessoa.** Nunca "a monografia do candidato" quando o texto é dele; escrever "minha monografia". Não deslizar para etiquetas distanciadas ao falar do próprio trabalho do autor.

3. **Overclaiming de vínculos causais/fundacionais.** Não escrever "fundamentam diretamente" para conexões que são indiretas. Ao justificar como a formação X sustenta o método Y, distinguir explicitamente conexões diretas (curso de métodos quantitativos → metodologia computacional) de indiretas (experiência profissional → familiaridade geral). Não agrupar itens tangenciais junto com os diretamente relevantes por conveniência retórica.

## Passe final obrigatório

Antes de apresentar qualquer versão "final": fazer uma passada dedicada procurando APENAS os três padrões acima, um de cada vez. Não esperar o Pedro apontar.

## Registro e convenções do projeto

- Português acadêmico brasileiro, primeira pessoa do singular quando o documento é do Pedro (ou plural majestático se o gênero do documento pedir; perguntar se ambíguo).
- Terminologia fixa: "Caixa de Conversão" (maiúsculas), "mil-réis", "Convênio de Taubaté", "padrão-ouro", eixo "ortodoxo ↔ expansionista" (na dissertação, "papelista" só em citação de época).
- Grafias de época preservadas apenas em citações diretas das fontes ("probidade do paiz", "attentado"); no corpo do texto, ortografia atual.
- Citações e referências: ABNT (autor-data). Referências novas entram também em `artigo/referencias.bib`.
- Números de validação sempre com a estatística nomeada: "κ de Cohen = 0,712", "ρ de Spearman = 0,670", vírgula decimal em português.
- Ao reportar método computacional: modelo com versão exata, data de execução, temperatura. Nunca "utilizou-se inteligência artificial" genérico.

## O que este projeto considera boa prosa metodológica

Frases declarativas curtas; uma afirmação, uma evidência. Limitações declaradas com número (taxa de falso negativo, κ por fase) em vez de advérbios defensivos ("relativamente robusto"). Nada de "cabe ressaltar que", "é importante destacar que", "nesse sentido" como conectivo vazio.
